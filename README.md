# Interview Routing Graphs

LangGraph/LangChain flows for routing interview messages through extraction, tooling, and interview loops.

## Requirements
- Python 3.12+
- `OPENROUTER_API_KEY`
- `ffmpeg`
- [`uv`](https://github.com/astral-sh/uv)

## Run
```
uv run python main.py
```

## Tests
```
uv run python -m pytest
```
